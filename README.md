# Career Recommendation System

A Machine Learning project that recommends top jobs based on **user skills, location, and experience level**.  
The backend uses **Python, Flask, and scikit-learn** (TF-IDF + cosine similarity) to rank jobs.  
The frontend uses **HTML, CSS, and JavaScript** to display job recommendations in a user-friendly interface.


## Features

- Recommends **top 5 relevant jobs** based on skills and location.
- Filters jobs by **city/location**.
- Ranks jobs by **skill similarity** using TF-IDF and cosine similarity.
- Simple frontend interface to input skills, location, and level.
- Clickable links to apply for jobs.
